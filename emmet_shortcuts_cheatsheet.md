# **Emmet shortcuts cheatsheet**

## id attribute: \#

### \#sidebar

```html
<div id="sidebar"></div>
```

## class attribute: \.

### \.container

```html
<div class="container"></div>
```

## Child: \>

### nav > ul > li

```html
<nav>
  <ul>
    <li></li>
  </ul>
</nav>
```

## Sibling: \+

### nav + div + p

```html
<nav></nav>
<div></div>
<p></p>
```

## Multiplication: \*

### ol > li \* 3

```html
<ol>
  <li></li>
  <li></li>
  <li></li>
</ol>
```

## Text: \{ }

### a{Click me}

```html
<a href="">Click me</a>
```

## Climb-up: \^

### div > p > span + em ^ bq

```html
<div>
  <p><span></span><em></em></p>
  <blockquote></blockquote>
</div>
```
